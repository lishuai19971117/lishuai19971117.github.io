实现碰撞检测

发表于 2018-06-30

一款游戏必然有它本身的规则才可以称为游戏，贪吃蛇的规则很简单，就是吃到“苹果”自身长度会加1，如果碰到边界、障碍或自身则游戏结束。那么如何来判断蛇与其它物体是否“碰撞”呢？这里我们可以根据坐标来实现。在整个页面上，每个对象都是有自己的坐标的，碰撞实际就是两者的横纵坐标相同了。代码如下：

if(newhead.col < 0 || newhead.col > 59)
        {
            pause();
            beginkey = false;
            gameOver();
        }
        if(newhead.row < 0 || newhead.row > 59)
        {
            pause();
            beginkey = false;
            gameOver();
        }

        for(var i = 0;i < this.body.length;i++)
        {
            if (this.body[i].col == newhead.col && this.body[i].row == newhead.row) 
            {
                pause();
                beginkey = false;
                gameOver();
            }
        }
        for( var i = 0;i < barrier.body.length;i++)
        {
            if (barrier.body[i].col == newhead.col && barrier.body[i].row == newhead.row) 
            {
                pause();
                beginkey = false;
                gameOver();
            }
        }
        this.body.unshift(newhead);

        if(newhead.col == apple.posX && newhead.row == apple.posY)
        {
            score = score + 100;
            while(true)
            {
                var checkApple = false;
                apple.posX = Math.floor(Math.random()*60);
                apple.posY = Math.floor(Math.random()*60);
                for(var i = 0;i < this.body.length;i++)
                {
                    if(apple.posX == this.body[i].col && apple.posY == this.body[i].row)
                        checkApple = true;
                }
                for(var i = 0;i < barrier.body.length;i++)
                {
                    if(apple.posX == barrier.body[i].col && apple.posY == barrier.body[i].row)
                        checkApple = true;
                }
                if(!checkApple)
                    break;
            }
        }
        else
        {
            this.body.pop();
        }
    };

可以看到，当蛇的头部坐标与障碍的坐标相同，则游戏失败，与“苹果”的坐标相同，则不用pop（）来删除最后一个对象，即长度+1。

实现贪吃蛇的控制

发表于 2018-06-30

实现贪吃蛇的控制我们需要用到上、下、左、右四个方向键，这里我们需要用到JQuery插件来实现键盘交互，对应的keyCode分别为37、38、39、40。

这里要注意蛇的运动变化只能是相当于蛇头前进方向向左或向右，而不能是向后。代码如下：

$("body").keydown(function(event)
    {
        console.log(event.keyCode);
        if(event.keyCode == 37 && snake.direction != "right")
        {
            snake.direction = "left";
        }
        if(event.keyCode == 38 && snake.direction != "down")
        {
            snake.direction = "up";
        }
        if(event.keyCode == 39 && snake.direction != "left")
        {
            snake.direction = "right";
        }
        if(event.keyCode == 40 && snake.direction != "up")
        {
            snake.direction = "down";
        }
    });

实现贪吃蛇的移动

发表于 2018-06-30

实现贪吃蛇的移动一个关键是屏幕刷新函数，但只有屏幕刷新是不够的，还需要自己来构造生成方块的函数。首先创建方块对象：

var Block = function(col,row,size)
    {
        this.col = col;
        this.row = row;
        this.size = size;
    };

三个参数分别是行、列坐标及大小，接下来创建snake这个对象：

var snake = {
        body: [
            new Block(20,20,10),
            new Block(20,21,10),
            new Block(20,22,10)
        ], 
        direction:"right",
    };

可以看出这条“蛇”是一个可以增加的方块数组，而direction参数决定蛇的移动方向，起始默认方向为向右。

对象创建完毕，接下来就是方法，有两个方法需要我们实现，一个是snake.draw()，另一个是snake.move()，代码如下：

snake.draw = function()
    {
        for(var i = 0;i < this.body.length;i++)
        {
            this.body[i].draw();
        }
    };

snake.move = function()
        {
            var head = this.body[0];
            if(snake.direction == "right")
            {
        var newhead = new Block(head.col + 1,head.row,head.size);
            }
        if(snake.direction == "left")
        {
            var newhead = new Block(head.col - 1,head.row,head.size);
        }
        if(snake.direction == "up")
        {
            var newhead = new Block(head.col,head.row - 1,head.size);
        }
        if(snake.direction == "down")
        {
            var newhead = new Block(head.col,head.row + 1,head.size);
        }

代码中创建了一个变量head，用来保存蛇头的对象。根据贪吃蛇运动的方向创建新的对象newhead，若方向为向右，则head.col+1，其他方向同理。通过unshift方法讲对象newhead添加到数组中。通过pop方法将数组中最后一个对象删除。

clearInterval的用法

发表于 2018-06-30

在了解了贪吃蛇移动的原理后，想在游戏中设置暂停的功能自然也就简单明了了，就是让setInterval的功能停止即可，那么这就要用到clearInterval语法了。

var begin = function()
    {
        if(beginkey)
        {
            intervalId = setInterval(func,time);
            beginkey = false;
        }

    };

    var pause = function()
    {
        clearInterval(intervalId);    
        beginkey = true;
    };

在代码中，我给setInterval赋予一个参数，然后直接执行clearInterval即可，如果想要结束暂停的话，就在执行一次setInterval。

setinterval的用法

发表于 2018-06-22

实现贪吃蛇移动的一个关键就是setInterval方法的使用，该方法的作用是按一定的时间间隔刷新页面。
setInterval格式如下：

setInterval(function,interval[,arg1,arg2,......argn])

其中function是调用的函数，后面的参数是时间间隔，单位是毫秒，例如我们可以设置setInterval（show（），100），那么页面就会按每100ms一次的频率刷新页面。小游戏中的代码如下：

var func = function()                
    {
        ctx.clearRect(0,0,600,600);
        ctx.textBaseline = "top";
        ctx.font = "20px Arial";
        ctx.fillText("你的分数："+score,12,5);
        snake.draw();
        snake.move();
        apple.draw();
        barrier.draw();
        ctx.strokeRect(0,0,600,600);

    };

    var begin = function()
    {
        if(beginkey)
        {
            intervalId = setInterval(func,time);
            beginkey = false;
        }

    };

func中有生成方块的函数，每执行一次生成一个方块，这样就可以实现贪吃蛇的移动了。

canvass的使用

发表于 2018-06-19

绘制方块

  <canvas id = "canvas" width = "400" height ="400"></canvas>
<script>
    var canv = docunment.getElementById("canvas");
    var ctx = canv.getContext("2d");
    ctx.fillRect(50,50,100,100);
</script>

该代码会生成一个100像素*100像素的方块，坐标在该画布的（50,50），颜色默认黑色，如果想改变颜色，可以在fillRect之前添加fillStyle函数.如：

ctx.fillStyle = "Red"；
ctx.fillRect(50,50,100,100);

即可使方块变成红色的，将多种方块巧妙地组合，可以得到很多有趣的东西，比如可以用黑白两种颜色的方块按规律排列便可以得到一张国际象棋棋盘。
