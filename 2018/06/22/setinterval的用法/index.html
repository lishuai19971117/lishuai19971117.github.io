setinterval的用法

发表于 2018-06-22

实现贪吃蛇移动的一个关键就是setInterval方法的使用，该方法的作用是按一定的时间间隔刷新页面。
setInterval格式如下：

setInterval(function,interval[,arg1,arg2,......argn])

其中function是调用的函数，后面的参数是时间间隔，单位是毫秒，例如我们可以设置setInterval（show（），100），那么页面就会按每100ms一次的频率刷新页面。小游戏中的代码如下：

var func = function()                
    {
        ctx.clearRect(0,0,600,600);
        ctx.textBaseline = "top";
        ctx.font = "20px Arial";
        ctx.fillText("你的分数："+score,12,5);
        snake.draw();
        snake.move();
        apple.draw();
        barrier.draw();
        ctx.strokeRect(0,0,600,600);

    };

    var begin = function()
    {
        if(beginkey)
        {
            intervalId = setInterval(func,time);
            beginkey = false;
        }

    };

func中有生成方块的函数，每执行一次生成一个方块，这样就可以实现贪吃蛇的移动了。
