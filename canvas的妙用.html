canvass的使用

绘制方块

  <canvas id = "canvas" width = "400" height ="400"></canvas>
<script>
    var canv = docunment.getElementById("canvas");
    var ctx = canv.getContext("2d");
    ctx.fillRect(50,50,100,100);
</script>

该代码会生成一个100像素*100像素的方块，坐标在该画布的（50,50），颜色默认黑色，如果想改变颜色，可以在fillRect之前添加fillStyle函数.如：

ctx.fillStyle = "Red"；
ctx.fillRect(50,50,100,100);

即可使方块变成红色的，将多种方块巧妙地组合，可以得到很多有趣的东西，比如可以用黑白两种颜色的方块按规律排列便可以得到一张国际象棋棋盘。
